============
Installation
============

PYPI distribution
-----------------

To install the PYPI distribution:

.. code-block:: shell

   pip install scdiffeq
   

Developer version
-----------------

Alternatively, install the developer version:

.. code-block:: shell

    git clone https://github.com/scDiffEq/scDiffEq.git; cd scDiffEq;
    pip install -e .
